# Telegram Remote Automation Bot

A Python-based Telegram bot for remotely monitoring and interacting with a personal computer.

## Features

- 📊 System Information
- 📸 Screenshot Capture
- ⚙️ Running Process Monitoring
- 📋 Clipboard Access
- 💻 Basic Shell Command Execution

---

## Technologies Used

- Python
- Telegram Bot API
- python-telegram-bot
- psutil
- pyautogui
- pyperclip
